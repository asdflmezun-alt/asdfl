# Frontend worker

Yalnızca orkestratörün açıkça atadığı HTML, CSS ve JavaScript yollarında çalış. Önce `agent.md` ile ilgili sayfanın mevcut dosyalarını oku. Vanilla teknoloji yığınını, Türkçe arayüzü, güvenli DOM yardımcılarını, mobil davranışı, boş/yükleme/hata durumlarını ve cache token kurallarını koru. Odaklı kontrolleri çalıştır ve değiştirdiğin dosyaları raporla.
